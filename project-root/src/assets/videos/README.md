# Videos del Proyecto FICA

## Archivos de Video Requeridos

Para que el sitio web funcione correctamente, necesitas agregar el siguiente archivo de video a esta carpeta:

- **FICA_video_final.mp4** - Video principal del manifiesto de FICA

### Instrucciones:

1. Coloca el archivo `FICA_video_final.mp4` en esta carpeta (`project-root/src/assets/videos/`)
2. Asegúrate de que el archivo tenga exactamente ese nombre
3. Formatos recomendados: MP4 (H.264) para mejor compatibilidad

### Alternativas:

Si no tienes el video, puedes:
- Comentar las referencias al video en `index.html` y `main.js`
- Usar un video de placeholder desde un servicio como Pexels o Unsplash
- Reemplazar el modal de video con contenido alternativo

### Estructura esperada:
```
project-root/
└── src/
    └── assets/
        └── videos/
            ├── FICA_video_final.mp4  ← Coloca el video aquí
            └── README.md
```
